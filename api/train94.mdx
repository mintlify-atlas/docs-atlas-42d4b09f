## Function Signature

```python
def train94(
    train_loader=None,
    epochs=9.9,
    label_smoothing=0.2,
    learning_rate=11.5,
    bias_scaler=64.0,
    momentum=0.85,
    weight_decay=0.0153,
    whiten_bias_epochs=3,
    tta_level=2,
    make_net=make_net94,
    run=0,
    verbose=True
)
```

## Description

Trains a convolutional neural network to achieve 94% accuracy on CIFAR-10. This is one of the fastest known algorithms for reaching this accuracy level, achieving 94.01% mean accuracy in under 10 epochs.

The network uses a wider architecture than train93, with network widths of 64/256/256 for blocks 1-3, providing more representational capacity while maintaining fast training speed.

## Parameters

<ParamField path="train_loader" type="CifarLoader" default="None">
  Data loader for CIFAR-10 training data. If None, a default loader is created with batch size 1024, flip augmentation, and translate augmentation of 2 pixels.
</ParamField>

<ParamField path="epochs" type="float" default="9.9">
  Number of training epochs. Can be fractional to train for partial epochs.
</ParamField>

<ParamField path="label_smoothing" type="float" default="0.2">
  Label smoothing factor for cross-entropy loss. Higher values (e.g., 0.2) smooth the target distribution.
</ParamField>

<ParamField path="learning_rate" type="float" default="11.5">
  Learning rate per 1024 examples. This is a decoupled learning rate that scales with batch size, allowing independent tuning from momentum and weight decay.
</ParamField>

<ParamField path="bias_scaler" type="float" default="64.0">
  Multiplier for learning rate applied to BatchNorm biases (but not weight decay). Allows BatchNorm biases to adapt faster.
</ParamField>

<ParamField path="momentum" type="float" default="0.85">
  Nesterov momentum coefficient. The decoupled parametrization ensures changing this doesn't require re-tuning the learning rate.
</ParamField>

<ParamField path="weight_decay" type="float" default="0.0153">
  Weight decay per 1024 examples (L2 regularization). Decoupled from learning rate to allow independent tuning.
</ParamField>

<ParamField path="whiten_bias_epochs" type="int" default="3">
  Number of epochs to train the whitening layer bias before freezing it. The whitening layer weights are always frozen.
</ParamField>

<ParamField path="tta_level" type="int" default="2">
  Test-time augmentation level for final evaluation:
  - 0: No augmentation
  - 1: Mirror flip augmentation
  - 2: Mirror + translate augmentation (most accurate)
</ParamField>

<ParamField path="make_net" type="callable" default="make_net94">
  Function that creates and returns the neural network model. Defaults to make_net94 which creates the standard 94% architecture.
</ParamField>

<ParamField path="run" type="int" default="0">
  Run identifier for logging purposes. Use -1 or 'warmup' for warmup runs that exit after the first batch.
</ParamField>

<ParamField path="verbose" type="bool" default="True">
  Whether to print training progress, including per-epoch metrics and final evaluation results.
</ParamField>

## Returns

<ResponseField name="model" type="torch.nn.Module">
  The trained PyTorch model in half precision (fp16) with channels_last memory format, ready for inference or evaluation.
</ResponseField>

## Performance Metrics

When run on an NVIDIA A100 GPU with the default hyperparameters:

- **Mean Accuracy**: 94.01% (measured over n=1000 runs)
- **Training Time**: 2.59s (using airbench94_muon.py variant with Muon optimizer)
- **Computational Cost**: 0.29 PFLOPs (airbench94_muon.py)
- **Legacy Time**: 3.83s (original version), 3.09s (compiled version)
- **Legacy PFLOPs**: 0.36 PFLOPs

<Note>
The fastest variant uses the Muon optimizer. The legacy versions in this library use standard SGD with Nesterov momentum.
</Note>

## Usage Example

```python
import airbench

# Basic usage with default hyperparameters
model = airbench.train94()

# Custom training with modified hyperparameters
train_loader = airbench.CifarLoader(
    'cifar10',
    train=True,
    batch_size=1024,
    aug={'flip': True, 'translate': 2}
)
model = airbench.train94(
    train_loader=train_loader,
    epochs=12.0,
    learning_rate=10.0,
    verbose=True
)

# Evaluate on test set
test_loader = airbench.CifarLoader('cifar10', train=False, batch_size=2000)
accuracy = airbench.evaluate(model, test_loader, tta_level=2)
print(f'Test accuracy: {accuracy:.2%}')

# Warmup run (exits after first batch)
airbench.warmup94()
```

## Architecture Details

The default network architecture uses:

- **Whitening layer**: 2x2 kernel preprocessing layer (weights frozen after initialization)
- **Block 1**: 64 channels
- **Block 2**: 256 channels (2x wider than train93)
- **Block 3**: 256 channels (2x wider than train93)
- **BatchNorm momentum**: 0.6
- **Activation**: GELU
- **Final scaling factor**: 1/9
- **Precision**: Half precision (fp16) training

## Data Augmentation

Default augmentation pipeline includes:

- **Horizontal flip**: Alternating flip scheme for increased diversity
- **Translation**: Random crops with 2 pixels of padding
- **No cutout**: Unlike train96, this configuration doesn't use cutout augmentation

## Comparison with Standard Training

For reference, standard ResNet-18 training achieves:
- **Mean Accuracy**: 96.0%
- **Training Time**: 7 minutes
- **Computational Cost**: 32.3 PFLOPs

This makes train94 approximately **160x faster** than standard training while achieving 94% accuracy.

## Data Selection Example

```python
import torch
from airbench import train94, infer, CifarLoader

# Train initial model without label smoothing
net = train94(label_smoothing=0)

# Get confidence scores
loader = CifarLoader('cifar10', train=True, batch_size=1000)
logits = infer(net, loader)
conf = logits.log_softmax(1).amax(1)

# Train on low-confidence examples
train_loader = CifarLoader('cifar10', train=True, batch_size=1024,
                          aug={'flip': True, 'translate': 2})
mask = (conf < conf.float().quantile(0.6))
train_loader.images = train_loader.images[mask]
train_loader.labels = train_loader.labels[mask]
model = train94(train_loader, epochs=16)
```
