## Current Records

CIFAR-10 Airbench contains the fastest known algorithms for training neural networks to 94% and 96% accuracy on CIFAR-10 using a single NVIDIA A100 GPU.

| Script | Mean Accuracy | Time | PFLOPs |
| ------ | ------------- | ---- | ------ |
| [airbench94_muon.py](https://github.com/KellerJordan/cifar10-airbench/blob/main/airbench94_muon.py) | 94.01% | 2.59s | 0.29 |
| [airbench96_faster.py](https://github.com/KellerJordan/cifar10-airbench/blob/main/airbench96_faster.py) | 96.00% | 27.3s | 3.1 |

<Note>
  Timings measured using `torch==2.4.1` on a 400W NVIDIA A100 GPU
</Note>

## Performance Improvements

### From Paper to Current Records

The current records represent significant improvements over the original paper results:

- **94% accuracy**: Improved from 3.09s to 2.59s (16% faster) using the Muon optimizer
- **96% accuracy**: Improved from 34.7s to 27.3s (21% faster) using data filtering techniques
- **Computational efficiency**: Reduced PFLOPs from 0.36 to 0.29 for 94% accuracy

### Key Innovations

<AccordionGroup>
  <Accordion title="Muon Optimizer">
    The [Muon optimizer](https://kellerjordan.github.io/posts/muon/) used in `airbench94_muon.py` provides faster convergence and improved training stability, enabling the current 94% accuracy speedrun record.
  </Accordion>
  
  <Accordion title="Data Filtering">
    Advanced data filtering techniques in `airbench96_faster.py` allow the model to train more efficiently by focusing on the most informative examples, reducing training time by 21%.
  </Accordion>
  
  <Accordion title="Compilation">
    PyTorch compilation (used in legacy scripts) provides modest speedups through graph optimization, though the impact is smaller than algorithmic improvements.
  </Accordion>
</AccordionGroup>

## Reproducing Results

To reproduce these benchmarks on your own hardware:

```bash
# Clone the repository
git clone https://github.com/KellerJordan/cifar10-airbench.git
cd cifar10-airbench

# Run the 94% accuracy record
python airbench94_muon.py

# Run the 96% accuracy record
python airbench96_faster.py
```

<Warning>
  Performance may vary based on GPU model, PyTorch version, and system configuration. The timings shown are specific to the hardware and software versions listed above.
</Warning>
