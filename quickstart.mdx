## Train Your First Model

Get from zero to a 94% accuracy CIFAR-10 model in just a few commands.

<Steps>
  <Step title="Clone the Repository">
    First, clone the Airbench repository:

    ```bash
    git clone https://github.com/KellerJordan/cifar10-airbench.git
    cd cifar10-airbench
    ```
  </Step>

  <Step title="Install Dependencies">
    Airbench requires PyTorch and torchvision. Install them if you haven't already:

    ```bash
    pip install torch torchvision
    ```

    <Note>
    Make sure you have a CUDA-compatible GPU and the appropriate CUDA toolkit installed. See the [installation guide](/installation) for detailed requirements.
    </Note>
  </Step>

  <Step title="Run the 94% Accuracy Script">
    Run the fastest training script to achieve 94% accuracy:

    ```bash
    python airbench94_muon.py
    ```

    This script will:
    - Download CIFAR-10 automatically (if needed)
    - Initialize the model with whitening
    - Train for approximately 8 epochs
    - Evaluate with test-time augmentation
  </Step>

  <Step title="View Results">
    You'll see output like this:


## Understanding the Training Script

Let's look at the key components of `airbench94_muon.py`:

### Model Architecture

The model uses a custom ConvNet with three convolutional groups:

```python
widths = dict(block1=64, block2=256, block3=256)
whiten_kernel_size = 2
whiten_width = 2 * 3 * whiten_kernel_size**2

self.whiten = nn.Conv2d(3, whiten_width, whiten_kernel_size, padding=0, bias=True)
self.layers = nn.Sequential(
    nn.GELU(),
    ConvGroup(whiten_width,     widths["block1"]),
    ConvGroup(widths["block1"], widths["block2"]),
    ConvGroup(widths["block2"], widths["block3"]),
    nn.MaxPool2d(3),
)
self.head = nn.Linear(widths["block3"], 10, bias=False)
```

### GPU-Accelerated Data Loading

Data is loaded directly to GPU and preprocessed there:

```python
train_loader = CifarLoader(
    "cifar10", 
    train=True, 
    batch_size=2000, 
    aug=dict(flip=True, translate=2)
)
```

### Muon Optimizer

The Muon optimizer applies orthogonalization to gradients:

```python
optimizer = Muon(filter_params, lr=0.24, momentum=0.6, nesterov=True)
```

## Using the Library API

You can also use Airbench as a library in your own code:

```python
import airbench

# Create data loaders
train_loader = airbench.CifarLoader(
    '/tmp/cifar10', 
    train=True, 
    aug=dict(flip=True, translate=4, cutout=16), 
    batch_size=500
)
test_loader = airbench.CifarLoader(
    '/tmp/cifar10', 
    train=False, 
    batch_size=1000
)

# Train a model to 94% accuracy
net = airbench.train94()

# Evaluate the model
logits = airbench.infer(net, test_loader)
accuracy = airbench.evaluate(net, test_loader)
print(f"Accuracy: {accuracy:.2%}")
```

## Data Selection Example

Airbench is great for rapid experimentation. Here's an example that demonstrates training on low-confidence examples:

```python
import torch
from airbench import train94, infer, evaluate, CifarLoader

# Train a model to get confidence scores
net = train94(label_smoothing=0)

# Get confidence scores for all training examples
loader = CifarLoader('cifar10', train=True, batch_size=1000)
logits = infer(net, loader)
conf = logits.log_softmax(1).amax(1)

# Create a loader with only low-confidence examples
train_loader = CifarLoader(
    'cifar10', 
    train=True, 
    batch_size=1024, 
    aug=dict(flip=True, translate=2)
)
mask = (conf < conf.float().quantile(0.6))
print(f'Training on {mask.sum()} low-confidence images')
train_loader.images = train_loader.images[mask]
train_loader.labels = train_loader.labels[mask]

# Train on selected examples
net = train94(train_loader, epochs=16)
accuracy = evaluate(net, test_loader)
print(f"Accuracy: {accuracy:.2%}")  # ~94% - better than random sampling!
```

<Tip>
This example runs in under 20 seconds on an A100, making it perfect for rapid experimentation.
</Tip>

## Customizing Training

You can modify the dataloader to experiment with different subsets:

```python
import airbench

# Create a loader
train_loader = airbench.CifarLoader(
    '/tmp/cifar10', 
    train=True, 
    aug=dict(flip=True, translate=4, cutout=16), 
    batch_size=500
)

# Filter to only certain classes (e.g., first 6 classes)
mask = (train_loader.labels < 6)
train_loader.images = train_loader.images[mask]
train_loader.labels = train_loader.labels[mask]

print(len(train_loader))  # Prints 60 (30,000 images / 500 batch size)
```

## Common Issues

<Warning>
If you encounter CUDA out of memory errors, try reducing the batch size in the training script.
</Warning>

<Accordion title="CIFAR-10 download fails">
  If the automatic download fails, you can manually download CIFAR-10 from [https://www.cs.toronto.edu/~kriz/cifar.html](https://www.cs.toronto.edu/~kriz/cifar.html) and place it in the `cifar10` directory.
</Accordion>

<Accordion title="Training is slower than expected">
  Make sure:
  - You're using an A100 GPU (other GPUs will be slower)
  - CUDA is properly installed and configured
  - You're using the correct PyTorch version with CUDA support
  - No other processes are using the GPU
</Accordion>
