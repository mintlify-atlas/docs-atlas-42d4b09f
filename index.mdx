  <Steps>
    <Step title="Clone the repository">
      Clone the CIFAR-10 Airbench repository to your local machine:
      
      ```bash
      git clone https://github.com/KellerJordan/cifar10-airbench.git
      cd cifar10-airbench
      ```
    </Step>
    
    <Step title="Install dependencies">
      Airbench requires PyTorch and torchvision. Install them using pip:
      
      ```bash
      pip install torch torchvision
      ```
      
      Or install the airbench package directly:
      
      ```bash
      pip install airbench
      ```
    </Step>
    
    <Step title="Run your first training">
      Run the 94% accuracy speedrun script:
      
      ```bash
      python airbench94_muon.py
      ```
      
      This will train a neural network to 94% accuracy on CIFAR-10 in approximately 2.6 seconds on an A100 GPU.
      
      <Note>
        Training times may vary depending on your GPU. The benchmarks are measured on a 400W NVIDIA A100 using PyTorch 2.4.1.
      </Note>
    </Step>
  </Steps>
